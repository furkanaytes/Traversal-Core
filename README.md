# Açıklama
- Traversal-Core
- .Net Core ile hazırlanan Seyahat sitesi. 
Bu proje vitrin, admin paneli ve kullanıcı paneli olmak üzere üç bölümden oluşmaktadır. 

# Kullanılan Teknolojiler
- ASP.NET Core MVC
- EntityFramework
- CodeFirst
- Linq
- Ajax
- Repository Design Pattern
- Fluent Validation
- Auto Mapper
- View Component
- N Tier Mimari
- Area
- Identity
- Login & Register
- Include Metotlar
- Api & Rapid Api & Api Consume
